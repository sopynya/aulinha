CSS

CSS (Cascading Style Sheets) é uma linguagem utilizada para definir a apresentação visual de documentos HTML, omo páginas web. Em outras palavras, o CSS controla como os elementos de uma página são exibidos, incluindo cores, fontes, espaçamento e layout.

Classes e ids

Dentro de um comando você pode nomea-lo usando " class="palavra" " ou " id="palavra" ". Classes podem ser usadas em multiplos comandos, e os id podem ser usadas apenas em um comando.

Exemplo:
<p class="palavrinha">Texto</>
<p id="textinho">Texto</>

A palavra "textinho" não poderá ser usada de novo para outro id, porém a palavra "palavrinha" pode ser usada em outros class.

Aplicar seu css

Para aplicar seu css no seu arquivo html basta usar o comando <link rel="style" href="nomedoseuarquivo.css">

Comandos css

Cor e fundo

color: muda cor;
background-color: cor de fundo;
background-image: imagem de fundo;

Fonte

font-size: tamanho das letras em pixels (px) ou tamanho padrão do navegador (em);
font-family: tipo de fonte;
font-weight: grossura da fonte, varia de 100 a 700;
text-align: alinhamento do texto;
text-decoration: decoração de texto ex:underline;
text-transform: deixa o texto em maiúsculo ou minúsculo (uppercase, lowercase);

Tamanho e espaçamento

width: largura;
height: altura;
padding: espaçamento interno;
margin: espaçamento externo;
max-width: largura máxima;

Borda

border: define cor, tipo e expessura da borda;
border-radius: arredontamento das bordas;
outline: borda de fora;

Posicionamento

display: determina o tipo de display(block, flex, none);
justify-content: alinha o conteudo horizontalmente caso o display seja flex;
align-items: alinha o conteudo verticalmente caso o display seja flex;
position: tipo de posição;
(top, right, bottom ou left): coordenadas da posição;

Mouse

cursor: tipo de cursor;
(comando, classe ou id):hover {seus comandos css} (muda ao passar o mouse por cima)
