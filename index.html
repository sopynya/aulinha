HTML

Html (HyperText Markup Language) é uma linguagem de marcação, é usada para estruturar o conteúdo visual de uma página.


Estrutura básica

A estrutura básica de um arquivo html são os comandos iniciais usados para sua página funcionar corretamente.

<!DOCTYPE html>

Este comando declara o tipo de arquivo, informa ao navegador que está usando a versão HTML5.

<html>

O comando raiz de todo arquivo html, todo conteúdo estará dentro desse comando.

<head>

Contém todas as informações que não aparecem visualmente.

<meta charset="UTF-8">

Define a codificação de caracteres (UTF-8 permite acentos e caracteres especiais corretamente).

<body>

Contém todo o conteúdo visível da página: textos, imagens, botões, links, etc.



IMPORTANTE!

Comandos que exigem texto fora dos "<" ">" devem ser finalizados com o mesmo comando, porém iniciando-se com "/".
Todo comando deve ter "<" e ">".

Comandos dentro do <body>

h1 até h6

São comandos para headings (títulos), o h1 é o maior e o h6 é o menor.

img

Comando para imagens, para botar uma imagem use "src="nomedaimagem.jpg"". Src também aceita links de imagem.

p

Comando para parágrafos de texto.

a

Comando para links, ao clicar você pode ser redirecionado para uma segunda página ou para uma seção da mesma página. Para redirecionar use "href="link ou seção"" depois do comando a.

Exemplo:
<a href="sites.com">palavra</a>

li

Comando para listas. Escreva cada item em comandos li separados.

ul

Agrupa os comandos li de forma não ordenada.

Exemplo:
<ul>
<li>item</li>
</ul>

ol

Lista ordenada (numerada). Agrupa os comandos li.

div

Container, agrupa diversos elementos.

span

Pequeno container, utilizado para ajustar estilo de textos.

button

Botão clicável.

input

Campo de entrada de dados (texto, email, senha, etc).

br

Pula uma linha.

hr

Cria uma linha horizontal, usada pra separar conteúdos.

video

Exibe um video.

audio

Toca um audio.

section

Funciona de forma parecida de um div, usado para agrupar seções de conteúdo.

header

Cabeçalho da página ou seção.

footer

Seção final de uma página.

form

Agrupa comandos input, serve como um formulário.

table

Cria uma tabela.

tr

Linha da tabela.

td

Célula de dado na tabela.

th

Cabeçalho de tabela.

nav

Menu de navegação, costuma abrigar comandos "a".

strong

Deixa o texto em negrito.

em

Deixa o texto em itálico.

aside

Conteúdo lateral.

Comandos dentro do <head>

style

Permite adicionar css diretamente no html.

script

Insere ou conecta arquivos JavaScript

link

Usado para conectar arquivos externos, como CSS, ícones e fontes.

title

Define o título.

meta

Fornece informações (metadados) como codificação, descrição, autor, etc.

base

Mostra um aviso para usuários que têm o JavaScript desativado.

Aplique seus conhecimentos

Faça um site que anuncia uma casa nova:

  • Inicie com uma estrutura básica

  • Adicione um título usando o comando title

  • Adicione uma descrição usando o comando <meta name="description" content="sua descrição">

  • Crie um header
  • Dentro do comando header, crie um h1 que diz "Casa"

  • Fora do header, crie um h2 que diz "Compre sua casa conosco!

  • Adicione uma imagem e use este link como src:
    https://images.adsttc.com/media/images/5d34/e507/284d/d109/5600/0240/newsletter/_FI.jpg?1563747560

  • Crie um seção usando o comando section
  • Dentro do comando section adicione um texto usando o comando p dizendo "Nossas casas são lindas"
  • Adicione uma lista com os seguintes beneficios da casa: "Grande", "Com piscina" e "Vista bonita".
